🖍 기능

1. 메인 페이지
+ 로그인
+ 회원가입
+ 리뷰 작성(로그인 해야 가능)
+ 프로필
    
2. 로그인
   + 회원 가입

3. 회원 가입
   + 아이디 중복 체크
   + 비밀번호 / 비밀번호 확인

4. 영화 리스트 (리뷰 작성)
   + 로그인 하면 리뷰 쓰기 버튼 생김
   + 영화 포스터 누르면 상세 설명 페이지로

5. 영화 상세 설명
   + 영화 줄거리
   + 평균 평점, 리뷰 목록
     
6. 리뷰 쓰기
   + 별점, 영화 평 등록

7. 리뷰 모음
   + 평점 순 오름,내림차순 정렬
  
8. 프로필
   + 가입했던 이름,이메일
   + 본인이 적었던 리뷰
